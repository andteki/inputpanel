# InputPanel

Title and input.

Az InputPanel példányosítása

```java
InputPanel inputPanel = new InputPanel();
```

Felirat beállítása

```java
inputPanel.setText("Teszt");
```

Felirat lekérdezése:

```java
String title = inpuPanel.getText();
```

Érték beállítása

```java
inputPanel.setValue("Teszt");
```

Érték lekérdezése:

```java
String value = inpuPanel.getValue();
```
